# DicionarioApp

O **DicionarioApp** é uma aplicação desktop desenvolvida em **C#** e **WPF** criada para auxiliar no registro e organização de vocabulário pessoal.

A proposta do projeto é funcionar como um dicionário pessoal e também como um caderno de anotações, permitindo adicionar novas palavras aprendidas, significados e exemplos de uso de forma prática e intuitiva.

A aplicação possui estrutura completa de **Front-end** e **Back-end**, utilizando armazenamento local em **JSON** para salvar os dados, garantindo funcionamento offline, leveza e facilidade de uso no dia a dia.

---

## Funcionalidades

- Cadastro de palavras, significados e exemplos
- Pesquisa rápida em tempo real
- Edição e remoção de registros
- Salvamento automático em JSON
- Funcionamento totalmente offline
- Interface moderna desenvolvida em WPF
- Estrutura organizada entre Front-end e Back-end

---

## Tecnologias Utilizadas

- C#
- .NET 8
- WPF
- JSON

---

## Objetivo

Projeto desenvolvido com foco em aprendizado, prática de desenvolvimento desktop moderno e criação de uma ferramenta útil para estudos e anotação de vocabulário.
