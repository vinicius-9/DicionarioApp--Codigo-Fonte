# 📖 DicionarioApp--Codigo-Fonte


> Repositório com o código-fonte completo do DicionarioApp.

---

## 📦 Quer usar sem compilar?

A versão pronta para usar está aqui:

👉 **[github.com/vinicius-9/Dicionario-Portable](https://github.com/vinicius-9/Dicionario-Portable)**

Baixe, extraia e use. Sem instalar nada.


---

## ✨ Funcionalidades

- Cadastro de palavras com significado e exemplo
- Busca rápida por palavra ou significado
- 60 palavras pré-cadastradas para já começar estudando
- Dados salvos localmente — sem internet, sem cadastro

---
## 🛠️ Como compilar

1. Baixe ou clone este repositório
2. Dê dois cliques em **`build.bat`**
3. O arquivo `saida\DicionarioApp_Portable.zip` será gerado automaticamente

**Requisitos para compilar:**
- Windows 10/11
- .NET 8 SDK *(o build.bat instala automaticamente se não tiver)*
